# ssm-admin
这是一个基于ssm的后台管理平台
## 运用到的技术
- Spring + Spring MVC + MyBatis 框架的整合及运用；
- MySQL 8 数据库的基本使用方法；
- Maven 的配置及使用；
- Tomcat 8 的配置及使用；
- Druid 数据库连接池；
- AJAX 异步技术；
- AdminLTE3、Bootstrap4、SweetAlert、JqGrid、jQuery 等前端框架组件及控件的使用；
- 多图上传技术；
- 大文件上传与文件的断点续传；
- 文件导入导出功能；
- Linux 系统部署及发布项目；
- Redis 缓存数据库的配置及使用方法；
- 前后端分离；
- Nginx 的配置及使用；
- Tomcat 集群的搭建及负载均衡；
- 使用 Nginx 实现动静分离部署；
- Java Web 性能优化的基本技巧。
